# OpenCart Manual Testing

Manual test report for [OpenCart](https://www.opencart.com) covering Registration, Login, and Dashboard modules.

## 🧪 Test Coverage

| Module | Test Scenarios | Test Cases |

|---|---|---|

| Registration Page | 1 | 20 |

| Login Page | 2 | 14 |

| Dashboard | 1 | 11 |

| **Total** | **4** | **45** |

## 📋 What's Tested

**Registration Page**

- Valid user registration

- Blank/empty field submissions

- Missing required fields (email, password, username, name, country)

- CAPTCHA/puzzle validation

**Login Page**

- Valid credentials login

- Empty credentials

- Invalid email / invalid password combinations

- Forgot password redirect

**Dashboard**

- Account details, Payment methods, Partner Up

- Showcase, Change password, Logout

- Orders, Downloads, Stores
